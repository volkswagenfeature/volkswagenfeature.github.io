Class: Package

Package(deploymentConfig, lock, parent, name, versionSpec, baseDir, production, sourcesCache)

A representation of an NPM package that is obtained from an external source, that may have dependencies on other packages, and may bundle packages in its node_modules/ sub folder.

Constructor

Creates a new package object.

baseDir String Directory in which the referrer's package.json configuration resides
production Boolean Indicates whether we deploy in production mode or development mode. In development mode, also the development dependencies will be included.
sourcesCache SourcesCache Cache that contains references to all sources that need to be obtained
